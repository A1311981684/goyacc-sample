# goyacc-sample

As of 1.8, 'go tool yacc' is no longer included. Instead, obtain 'goyacc' with the following: 
```go get -u golang.org/x/tools/cmd/goyacc```

Followed by: 
```goyacc -o calc.go -p Calc calc.y```
